# Welcome to the CHOW BUDDYS repo

## Setup this repo

1. Clone
2. run `npm i`
3. run `npm start`
4. Instructions should pop up in the browser to open in prefered device

## Colour Pallet

- Brown: `#C6A477`
- Goldy Brown: `#ECD59F`, `rgb(236, 213, 159)`
- Sky Blue: `#D3E7EE`
- Blue: `#ABD1DC`
- Dark Blue: `#7097A8`
